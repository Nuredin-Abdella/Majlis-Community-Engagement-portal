A comprehensive web portal for the Ethiopian Islamic Affairs Supreme Council (EIASC), serving as the digital gateway to Ethiopia's highest Islamic authority.

The Majlis Community Engagement Portal is a modern, responsive website designed to connect the Ethiopian Muslim community with the Ethiopian Islamic Affairs Supreme Council. With over 60 years of service, the EIASC represents Ethiopian Muslims both domestically and internationally, and this portal serves as the primary digital platform for community engagement, information dissemination, and service delivery.

## ğŸ“‹ About the Organization

The Ethiopian Islamic Affairs Supreme Council (Majlis) is:
- **Established**: April 6, 1975 (Rabiul Awal 12, 1395 A.H.)
- **Status**: Country's highest Islamic authority and public institution (reinstated by decree No. 1207/2020)
- **Mission**: To create a responsible, organized, conscientious, and competent Muslim community with a higher spiritual, social, and economic life
- **Vision**: To see a vibrant Ethiopian Muslim community living Islam in accordance with Allah's will, contributing positively to national harmony and development

## ğŸš€ Getting Started

### Prerequisites
- A modern web browser (Chrome, Firefox, Safari, Edge)
- A local web server (optional, for development)

### Installation

1. **Clone the repository**
   ```bash
   git clone https://github.com/yourusername/Majlis-Community-Engagement-portal.git
   cd Majlis-Community-Engagement-portal
   ```

2. **Open in browser**
   - Simply open `index.html` in your web browser
   - Or use a local server:
     ```bash
     # Using Python 3
     python -m http.server 8000
     
     # Using Node.js (http-server)
     npx http-server
     ```

3. **Access the website**
   - Direct file: `file:///path/to/index.html`
   - Local server: `http://localhost:8000`
